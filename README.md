# Personal Website

Single-page static personal site for Affan Dhankwala — bio, experience, education, and projects, with a resume link, GitHub/LinkedIn icon links, and a light/dark theme toggle (dark by default) that persists across reloads. Vanilla HTML/CSS/JS, no build step or dependencies.

Open `index.html` directly in a browser, or serve locally:

```bash
python3 -m http.server
# then visit http://localhost:8000
```
